# HeadlessTracker — Market & competitive intel

What I learn about the market, users, competitors. Not append-only — restructure as understanding evolves.

## Competitors (initial snapshot — verify on first scan)

- **Coinglass** — paid dashboard, derivatives-focused. Strong open interest / funding rate data. No MCP/programmatic angle.
- **Whale Alert** — large-transaction tracker, mostly Twitter alerts. Not portfolio.
- **DefiLlama** — DeFi TVL / yield data. Public API. Not portfolio per se.
- **CoinTracker / Koinly / Accointing** — paid tax-focused portfolio trackers. Closed-source.

**Our differentiator hypothesis**: MCP-native (LLM-callable), open-source, multi-source (exchanges + chains + prediction markets in one). To verify: are there users actually pairing this with Claude/Cursor/whatever?

## Target ICP (initial guesses — refine as Hex talks to people)

- **Power-user crypto traders** with multiple exchange accounts + on-chain positions. They currently use 3-5 dashboards. They want one query surface.
- **AI/agent builders** (LangChain, n8n, Cursor users) who want crypto data accessible via MCP without writing their own scrapers.
- **Tax/portfolio reporters** — maybe — if we add cost-basis tracking.

- X engagement on Hex's posts — engineering audience reach
- HN/Reddit hits if any (rare, treat as signal not strategy)
