Listázd ki a hub elérhető slash command-jait **Markdown szövegben** (NE card-ban — info-only, gomb nem kell). Üres sor a szekció-fejek között.

```
*📋 Hub parancsok*

Telegram-on a `/` gombbal autocomplete. Argumentum a parancs után space-szel.

*🎯 Munkanap kezdés*
• `/fokusz` — pillanatkép mindenből, darálási sorrend

*🔄 Riportok / pillanatkép*
• `/projektek` — projekt-státusz
• `/teendok` — TickTick 7 nap
• `/naptar` — ma + holnap

*⚡ Cron-ok ad-hoc*
• `/email` — email-check
• `/hirek` — napi digest
• `/edzo` — reggeli edző-riport

*🤖 Agentek*
• `/worker` — háttér-worker állapot, `/worker log` 24h log, `/worker <üzenet>` delegálás
• `/stokes` — Szandra-asszisztens állapot, `/stokes log` 24h üzenet-történet, `/stokes <üzenet>` delegálás (pl. `/stokes szólj a feleségemnek hogy fél órát késem`)

*🔍 Egyéb*
• `/wiki <query>` — wiki keresés
• `/szia` — bemutatkozás
• `/help` — ez a lista
```
